Tokenizer

token：最小语义单元。多模态模型：图像等 -> token，再与文本的 token 进行拼接。

tokenizer：分词器。输入文本 -> token_id（input_id）。

input ids：token 的索引，是整数向量。

tokenizer.decode()：将 input_ids 转化为对应的 embedding 向量。
