Agent 与 LangGraph 协作进行结构化推理
使用 Azure Cosmos DB 进行实时聊天历史记录存储 - 该数据库为历史上增长最快的 AI 代理应用程序 ChatGPT 中的聊天历史记录提供支持

FastAPI 的异步功能可保持交互响应灵敏且可扩展
在本课程结束时，您将拥有一个清晰的蓝图，用于构建和部署您自己的可扩展、云原生多代理应用程序，这些应用程序利用现代人工智能和云基础设施的力量。无论您是人工智能工程师、云架构师还是 Python 开发人员，本次演讲都将为您提供实用的见解和经过实战考验的模式，以构建下一代人工智能驱动的应用程序
