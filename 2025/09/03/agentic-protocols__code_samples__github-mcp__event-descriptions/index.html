Agent 与 LangGraph 协作进行结构化推理
使用 Azure Cosmos DB 进行实时聊天历史记录存储 - 该数据库为历史上增长最快的 AI 代理应用程序 ChatGPT 中的聊天历史记录提供支持
使用 Cosmos DB 的本机嵌入支持进行知识检索的矢量搜索
FastAPI 的异步功能可保持交互响应灵敏且可扩展
在本课程结束时，您将拥有一个清晰的蓝图，用于构建和部署您自己的可扩展、云原生多代理应用程序，这些应用程序利用现代人工智能和云基础设施的力量。无论您是人工智能工程师、云架构师还是 Python 开发人员，本次演讲都将为您提供实用的见解和经过实战考验的模式，以构建下一代人工智能驱动的应用程序

活动名称：提示是新的脚本：认识 GenAIScript

描述

jQuery 曾经通过抽象复杂性使 Web 开发变得更加容易，使开发人员能够专注于构建而不是与浏览器怪癖作斗争。如今，人工智能的发展面临着类似的挑战。新的模式不断出现，要跟上它可能会让人不知所措，尤其是当人工智能工具（尤其是代理工具）变得更加强大和复杂时。如果您可以利用尖端的人工智能功能，使用简单、熟悉的 JavaScript 抽象来自动执行任务，结果会怎样？ GenAIScript 是一种将 AI 轻松集成到工作流程中的方法，将提示视为可重用的代码片段。在本次演讲中，我们将探讨 GenAIScript 如何让 AI 自动化代理像编写 JavaScript 一样直观，帮助您简化重复性工作，而无需深厚的 AI 专业知识。
