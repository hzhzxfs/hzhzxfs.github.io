关键要点总结

  1. async def：定义协程函数
  2. await：挂起当前协程，等待其他操作完成
  3. Future：代表异步操作的最终结果
  4. Task：包装协程，用于并发执行
  5. 事件循环：管理和调度所有异步任务

异步编程的核心思想是：在等待I/O操作时让出控制权，而不是阻塞线程，从而实现高效的并发处理。
