TaskOutputTool：读取任务输出

它把后台任务输出统一成同一种可读结果

TaskOutputTool 用来读取后台任务的输出。
它最重要的价值是统一：

  • shell 任务输出
  • 本地 agent 输出
  • 远程 agent 输出

所以主线程不需要关心底层任务类型差异，只要关心统一返回结构。
