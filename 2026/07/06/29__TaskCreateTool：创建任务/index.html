TaskCreateTool：创建任务

它把待办项升级成正式任务对象

TaskCreateTool 不是简单往列表里插一行文本，而是把一个工作项创建成正式任务对象：

  • 有 id
  • 有 subject
  • 有 description
  • 有状态
  • 可被后续更新、阻塞、归属

这说明 Claude Code 的任务系统已经不是展示层小功能，而是真正的运行时对象系统。
